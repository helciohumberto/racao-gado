# 🐄 racao-gado

Sistema de gestão de pecuária desenvolvido por um desenvolvedor com raízes no agronegócio.

## Motivação

Meu tio gerencia uma fazenda no Brasil com gado de corte. Vi de perto os desafios do dia a dia: controle de pesagens, custo de ração, histórico de cada animal. Tudo era feito no papel ou em planilhas simples. Quando aprendi a programar, a primeira coisa que quis construir foi uma solução para esse problema real — que eu conheço de perto.

## O que este sistema resolve

- Cadastro e histórico de animais
- Registro de pesagens e cálculo de GMD (Ganho Médio Diário)
- Controle de ração: consumo e custo por cabeça/dia
- Dashboard com métricas de produtividade
- Relatórios de custos por período

## Stack

**Backend:** Node.js · Express · TypeScript · Prisma ORM · PostgreSQL  
**Frontend:** React · Vite · TypeScript · Tailwind CSS · Recharts  
**Auth:** JWT  
**Deploy:** Railway (backend + DB) · Vercel (frontend)
